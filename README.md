## 📌 About the Project

This is my **Personal Portfolio Website** built to showcase my skills, projects, and contact information.
It serves as a professional online presence where recruiters and visitors can explore my work and learn more about my development journey.

The portfolio highlights my major projects including **HelpMe** and **Everything Mall**, along with my technical skills and experience.

## 🚀 Getting Started (Run Locally)

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Manish-pro854/Potpolio.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Potpolio
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
